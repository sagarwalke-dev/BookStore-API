Welcome in Book Store API

Endpoint Description Example
/getAllBooks It return all books data from database http://localhost:3000/getAllBooks
/getBookById It return book data by id from database
bookId : 123 Book id which we want to retrive data		 http://localhost:3000/getBookById?bookId=123
/addBook It add the book data into database http://localhost:3000/addBook?bookName=Nodejs&bookAuthor=prashanth&bookRelase=1997-09-03&bookPrice=129&bookQty=20
/deleteBookById It delete the book data by Id http://localhost:3000/deleteBookById?bookId=2
/updateBookQtyById It update the book Qty by the given Id http://localhost:3000/updateBookQtyById?bookId=3&bookQty=20
